Glosario de comandos GIT

Comando Breve Descripción Parámetros Descripción de los Parámetros Ejemplo
git init Inicializa un repositorio Git Sin parametros Crea un nuevo repositorio Git en el directorio actual. git init
git clone Clona un repositorio Git existente URL del repositorio Copia un repositorio Git remoto en tu máquina local. git clone https://github.com/Hombrux/u1-ejerciciogit
git add Añade cambios al área de preparación Nombre de archivo(s) Añade archivos modificados o nuevos al área de preparación para ser confirmados. git add archivo.txt
git commit Registra cambios en el repositorio -m "Mensaje de confirmación" Guarda los cambios confirmados en el historial del repositorio. git commit -m "Agrega archivo.txt"
git status Muestra el estado de los archivos en el repositorio Sin parametros Muestra el estado actual de los archivos en el repositorio, incluyendo cambios sin confirmar. git status
git log Muestra el historial de confirmaciones Sin parametros Muestra el historial de confirmaciones en el repositorio, incluyendo información sobre autores y fechas. git log
git branch Muestra la lista de ramas -r para ver ramas remotas Muestra la lista de ramas en el repositorio. Con la opción -r, muestra ramas remotas. git branch
git checkout Cambia de rama o restaura archivos Nombre de la rama o -- nombre del archivo Permite cambiar de rama o restaurar archivos desde el historial. git checkout nombre_de_rama
